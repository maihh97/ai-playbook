# GPT Examples

This repository contains lightweight transformer and embedding examples. 
- small GPT-style model artefacts
- tokenisation experiments (standard and BPE)
- encoder-decoder notebook experiments
- an embeddings and retrieval walkthrough notebook

Generated by GitHub Copilot.

## Purpose

This repository is designed as a practical learning space for understanding what's behind modern LLMs. It's *NOT* used for production/ medical settings.

## Requirements

Typical Python requirements for the notebooks:
- Python 3.10+
- torch
- numpy
- matplotlib
- scikit-learn
- jupyter

Install with:

```bash
pip install torch numpy matplotlib scikit-learn jupyter
```

## Getting Started

1. Open the repository in VS Code or Jupyter Lab.
2. Create and activate a virtual environment.
3. Install dependencies.
4. Open a notebook and run cells in order.

Suggested reading order:
1. `mini_gpt2_character_tokenisation.ipynb`
2. `mini_gpt2_bpe_tokenisation.ipynb`
3. `embeddings_explainer.ipynb`
4. `use_trained_model.ipynb`
